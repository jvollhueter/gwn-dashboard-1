# API-Referenz

Die API-Referenz wird mit **mkdocstrings** unmittelbar aus den englischen Docstrings des Python-Pakets `gwn_dashboard` erzeugt.

Die Navigation ist nach den technischen Schichten der Anwendung gegliedert:

- Anwendungskomposition und Konfiguration
- Domain-Modelle
- Repositories
- Services
- Visualisierung und Design
- Benutzeroberfläche

Die API-Seiten dokumentieren öffentliche Klassen, Funktionen, Methoden, Parameter, Rückgabewerte und relevante Ausnahmen. Fachliche Erläuterungen und Bedienungsanleitungen befinden sich in den eigenständigen Dokumentationsbereichen.
