# Glossar

| Begriff | Bedeutung |
|---|---|
| API-Referenz | automatisch aus Code und Docstrings erzeugte technische Dokumentation |
| Cache | Zwischenspeicher zur Vermeidung wiederholter Berechnungen oder Dateizugriffe |
| CDC | hier nicht als aktive Datenquelle verwendet; allgemeine Abkürzung für Climate Data Center |
| Detail-GWK | für Detaildiagramme und Kennzahlen ausgewählter Grundwasserkörper; keine Untereinheit |
| ETp | potenzielle Evapotranspiration |
| GeoDataFrame | tabellarische Datenstruktur mit Geometriespalte und Koordinatenreferenzsystem |
| GWK | Grundwasserkörper |
| GWK25 | Zuordnung zum Gebietsstand GWK 2025 |
| GWN | Grundwasserneubildung |
| KLiWES | Bezeichnung des zugrunde liegenden Modell-/Datenkontexts |
| MKZ | Messstellenkennzahl |
| Nomogramm | in der Anwendung explorative grafische Beziehung zwischen GWN und Niederschlag |
| MkDocs Material | Dokumentationssystem für die fachlichen Seiten, das Benutzerhandbuch und die Navigation |
| mkdocstrings | Erweiterung zur automatischen Erzeugung der API-Referenz aus Python-Docstrings |
| P | Niederschlag |
| Referenzzeitraum | früherer Zeitraum, dessen Statistik als Vergleichsbasis dient |
| rg1 | Zufluss zum schnellen Grundwasserspeicher |
| rg2 | Zufluss zum langsamen Grundwasserspeicher |
| Session State | sitzungsbezogener Streamlit-Zustand für UI-Auswahlen |
| Vergleichszeitraum | späterer Zeitraum, dessen Statistik mit der Referenz verglichen wird |
